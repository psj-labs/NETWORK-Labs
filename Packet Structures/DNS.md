# DNS Header Structure (12 bytes)

```text
############################################################
                      DNS Header (12 bytes)
############################################################

┌─────────────────────────────────────────────────────────┐
│                    ID  (16 bits, Query ID)              │
└─────────────────────────────────────────────────────────┘

┌────┬─────────┬────┬────┬────┬────┬───────┬──────────────┐
│ QR │ Opcode  │ AA │ TC │ RD │ RA │   Z   │    RCODE     │
│1b  │ 4 bits  │1b  │1b  │1b  │1b  │ 3bits │   4 bits     │
└────┴─────────┴────┴────┴────┴────┴───────┴──────────────┘

┌─────────────────────────────────────────────────────────┐
│                    QDCOUNT  (16 bits)                   │
└─────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────┐
│                    ANCOUNT  (16 bits)                   │
└─────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────┐
│                    NSCOUNT  (16 bits)                   │
└─────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────┐
│                    ARCOUNT  (16 bits)                   │
└─────────────────────────────────────────────────────────┘


ID (16 bits)
 : DNS Query / Response 매칭 식별자
   예시 → 0x1A2B

QR (1 bit)
 : 패킷 타입 구분
   - 0 → Query
   - 1 → Response
   예시 → 0

Opcode (4 bits)
 : 질의 종류
   - 0 → Standard Query
   - 1 → Inverse Query (폐기)
   - 2 → Server Status
   예시 → 0

AA – Authoritative Answer (1 bit)
 : 권한 있는 네임서버 응답 여부
   예시 → 0

TC – Truncation (1 bit)
 : 메시지 길이 초과로 패킷이 잘렸는지 여부
   예시 → 0

RD – Recursion Desired (1 bit)
 : 클라이언트의 재귀 질의 요청 여부
   예시 → 1

RA – Recursion Available (1 bit)
 : 서버가 재귀 질의 지원 여부
   예시 → 1

Z (3 bits)
 : 예약 필드, 항상 000

RCODE (4 bits)
 : 응답 상태 코드
   - 0 → No Error
   - 3 → Name Error (NXDOMAIN)
   예시 → 0

QDCOUNT (16 bits)
 : Question 영역의 항목 개수
   예시 → 1

ANCOUNT (16 bits)
 : Answer 영역 포함 RR 개수
   예시 → 1

NSCOUNT (16 bits)
 : Authority 영역 RR 개수
   예시 → 0

ARCOUNT (16 bits)
 : Additional 영역 RR 개수
   예시 → 0
