# UDP Header Structure (8 bytes)

```text
################################################################################
                               UDP Header (8 bytes)
################################################################################

┌──────────────────────────┬──────────────────────────┐
│        Source Port       │      Destination Port    │
│          (2 bytes)       │         (2 bytes)        │
└──────────────────────────┴──────────────────────────┘

┌──────────────────────────┬──────────────────────────┐
│           Length         │           Checksum       │
│          (2 bytes)       │           (2 bytes)      │
└──────────────────────────┴──────────────────────────┘


Source Port (2 bytes)
 : 출발지 포트
   예시 → 53

Destination Port (2 bytes)
 : 목적지 포트

Length (2 bytes)
 : UDP 헤더(8B) + 데이터 전체 길이
   예시 → 48 bytes

Checksum (2 bytes)
 : UDP 전송 무결성 검증 값
   - 계산 범위 → 의사헤더 + UDP 헤더 + UDP 데이터
   - IPv4 → 옵션 (0 허용)
   - IPv6 → 필수
