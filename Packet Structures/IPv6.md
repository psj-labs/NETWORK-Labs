# IPv6 Header Structure (Fixed 40 bytes)

```text
################################################################################
                              IPv6 Header (40 bytes)
################################################################################

┌───────────────┬──────────────────────────────────────────────────────────────┐
│  Version (4)  │              Traffic Class (8) / Flow Label (20)             │
└───────────────┴──────────────────────────────────────────────────────────────┘

┌──────────────────────────┬────────────────┬───────────────────────────────────┐
│      Payload Length      │   Next Header  │            Hop Limit              │
│        (2 bytes)         │    (1 byte)    │             (1 byte)              │
└──────────────────────────┴────────────────┴───────────────────────────────────┘

┌──────────────────────────────────────────────────────────────────────────────┐
│                           Source IPv6 Address (16 bytes)                     │
└──────────────────────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────────────────────┐
│                        Destination IPv6 Address (16 bytes)                   │
└──────────────────────────────────────────────────────────────────────────────┘


Version (4 bits)
 : IPv6 버전 번호
   예시 → 6

Traffic Class (8 bits)
 : 서비스 클래스 / QoS 구분 값
   예시 → 0

Flow Label (20 bits)
 : 동일 플로우 식별자
   예시 → 0x00001

Payload Length (2 bytes)
 : IPv6 헤더 뒤에 이어지는 페이로드 전체 길이
   예시 → 1200

Next Header (1 byte)
 : 다음 프로토콜 또는 확장 헤더 식별자
   예시 → 6 (TCP)
           17 (UDP)
           58 (ICMPv6)

Hop Limit (1 byte)
 : TTL 역할 (홉마다 1씩 감소)
   예시 → 128

Source IPv6 Address (16 bytes)
   예시 → fe80::1ff:fe23:4567:890a

Destination IPv6 Address (16 bytes)
   예시 → fe80::2aa:bbff:fecc:ddee
