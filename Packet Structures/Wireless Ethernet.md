# IEEE 802.11 (Wireless Ethernet) Frame Structure

```text
####################################################################################################
                                  무선 Ethernet 프레임 (IEEE 802.11)
####################################################################################################

┌────────────┬──────────────┬────────────────┬────────────────┬────────────────┬──────────────────┬────────────────┐
│ Frame Ctrl │ Duration/ID  │     Addr1      │     Addr2      │     Addr3      │   Seq Control    │     Addr4*     │
│  (2 bytes) │   (2 bytes)  │    (6 bytes)   │    (6 bytes)   │    (6 bytes)   │     (2 bytes)    │    (6 bytes)   │
└────────────┴──────────────┴────────────────┴────────────────┴────────────────┴──────────────────┴────────────────┘
┌──────────────────────────────────────────────────────────────────────────────────────────────────┐
│                                      QoS Control (2 bytes)*                                      │
└──────────────────────────────────────────────────────────────────────────────────────────────────┘
┌──────────────────────────────────────────────────────────────────────────────────────────────────┐
│                                      HT Control (4 bytes)*                                       │
└──────────────────────────────────────────────────────────────────────────────────────────────────┘
┌──────────────────────────────────────────────────────────────────────────────────────────────────┐
│                                     Frame Body (0 ~ 2312 bytes)                                  │
│                                  ←────────── Payload ──────────→                                 │
└──────────────────────────────────────────────────────────────────────────────────────────────────┘
┌──────────────────────────────────────────────────────────────────────────────────────────────────┐
│                                                 FCS                                              │
└──────────────────────────────────────────────────────────────────────────────────────────────────┘

* Addr4, QoS Control, HT Control 필드는 특정 조건일 때만 존재함.

Frame Control (2 bytes)
 : 프레임 타입 및 서브타입, To DS / From DS 등 동작 제어 플래그 포함
   예시 → 08 02

Duration / ID (2 bytes)
 : NAV(Network Allocation Vector) 타이머 또는 Power Save AID 표시
   예시 → 3A 01

Addr1 (6 bytes) – Receiver Address
 : 프레임을 수신할 장치 주소
   예시 → ff:ff:ff:ff:ff:ff

Addr2 (6 bytes) – Transmitter Address
 : 프레임 송신 장치 주소
   예시 → 08:00:27:12:34:56

Addr3 (6 bytes) – BSSID / Destination Address
 : BSS 식별자 또는 최종 목적지 주소
   예시 → 9c:ef:d5:12:aa:20

Seq Control (2 bytes)
 : Sequence Number + Fragment Number
   예시 → 30 00

Addr4 (6 bytes) *
 : To DS = 1 AND From DS = 1 일 때 존재 (WDS 브릿지 사용 시)
   예시 → 00:25:90:ab:cd:ef

QoS Control (2 bytes) *
 : QoS 트래픽 제어 정보 필드 (우선순위, 큐 관리 등)
   예시 → 00 03

HT Control (4 bytes) *
 : 고속 전송 제어 정보 (802.11n 이상 PHY 제어)
   예시 → 00 00 00 00

Frame Body (Payload) (0 ~ 2312 bytes)
 : 실제 데이터 영역
   예시 → [IP Header + TCP Data + HTTP Request]

FCS – Frame Check Sequence (4 bytes)
 : CRC32 오류 검출 필드
   예시 → A1 B2 C3 D4
